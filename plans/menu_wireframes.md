# Вайрфреймы меню умного дома

## Допущения
- Устройство: стационарный 10" экран 1280×800 px, рабочая область 12 колонок × 160 px с базовым вертикальным модулем 80 px.
- Стиль: стеклянные панели (фон rgba(255,255,255,0.12) поверх размытых бэкграундов), мягкие тени 0 8 24 rgba(0,0,0,0.25), радиус 20 px.
- Шрифт: Manrope (regular 16 px, medium 18 px, semibold 22–28 px). Цвет текста — #F4F7FF (основной) и #9FB2D6 (вторичный).
- Верхняя горизонтальная панель меню фиксирована, высота 96 px, содержит иконки разделов + индикаторы состояния.
- Скролла нет: каждый экран компонуется в пределах 1280×800, учитывая 24 px внешних отступов.

### Вайрфрейм
- Фон: градиент #0A1A2F→#101C2E, стеклянные панели с размытием 20 px, обводка rgba(255,255,255,0.08).
- Иконки: монохром #F4F7FF, активные — с подсветкой неона #74F9FF.
- SOS-кнопка — круглая стеклянная красная (#FF4D67, opacity 0.85) с мягким свечением.
- Предупреждения на графике энергии — оранжевые ярлыки с треугольником.
- Анимации: 200 мс fade/slide для появления тултипов, пульсация активных сцен (opacity 0.8→1.0).

### Сценарии использования
- Пользователь быстрым тапом включает сцену «Домой», ожидает мгновенной смены подсветки карточки.
- Просмотр статуса качества воздуха; при тревоге карточка окрашивается в #FFB347 и предлагает открыть окно.
- Анализ энергопотребления: тап по пику открывает модал с деталями по устройствам.
